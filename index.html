🔑 Hướng dẫn lấy Google Gemini API Key

Để sử dụng trợ lý tuyển sinh, bạn cần có một API Key từ Google (hoàn toàn miễn phí). Vui lòng làm theo các bước sau:

  1. Truy cập trang web Google AI Studio.
  2. Đăng nhập bằng tài khoản Google (Gmail) của bạn.
  3. Nhấn vào nút "Get API key" ở góc trên bên trái.
  4. Nhấn "Create API key", chọn dự án của bạn (hoặc tạo dự án mới) rồi chọn "Create API key in new project".
  5. Sao chép (Copy) mã API Key vừa tạo (mã bắt đầu bằng AIzaSy...).
  6. Dán mã vào ô nhập API Key ở cột bên trái của ứng dụng này và nhấn "Lưu Key".

⚠️ Lưu ý: Tuyệt đối không chia sẻ API Key này cho người khác để tránh bị giới hạn hoặc vô hiệu hóa khóa.
